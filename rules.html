Server Rules

Please follow these rules to keep TitanNetwork fair, competitive, and enjoyable for everyone.

AFK Mining & Macros
Using macros, scripts, or unattended farming/mining methods is not allowed.
Cheating / Hacked Clients
Any unfair advantage tools (reach, killaura, xray, fly, etc.) are strictly forbidden.
Scamming & Theft
Scamming players, fake trades, or stealing from trusted islands is prohibited.
Chat Conduct
No harassment, hate speech, threats, or excessive spam in chat or private messages.
Bug / Exploit Abuse
Report bugs instead of abusing them for items, money, or progression.
Alt Account Abuse
Reasonable alt usage is allowed, but using alts to bypass punishments or economy rules is forbidden.
Respect Staff Decisions
If you disagree with a punishment, use the appeal page instead of arguing in public chat.
